## Dev instrucciones

El admin esta dividido en la API y el front-end con sus respectivas carpetas

#### Instrucciones
1.  Clonar el repositorio 
2. cd api/
3. `npm install`
4. Agregar el archivo .env
5. `npm run dev`
6. cd ../front-end/
7. `npm install`
8. `npm run dev`
